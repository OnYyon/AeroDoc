**Вопрос:** Как работают циклы в программировании?

**Ответ:** Циклы — это конструкции, которые позволяют повторять выполнение блока кода определенное количество раз или до тех пор, пока не будет достигнута какая-то условность. В Python есть три основных типа циклов:

1. **Цикл for**: используется для перебора элементов в коллекции (например, списке или кортеже). Синтаксис:
```python
for переменная in коллекция:
    # код, который нужно выполнить для каждого элемента
```
Пример:
```python
fruits = ['яблоко', 'груша', 'банан']
for fruit in fruits:
    print(fruit)
```
2. **Цикл while**: используется для повторения блока кода до тех пор, пока не будет достигнута какая-то условность. Синтаксис:
```python
while условие:
    # код, который нужно выполнить, пока условие истинно
```
Пример:
```python
i = 0
while i < 5:
    print(i)
    i += 1
```
3. **Цикл range**: используется для создания последовательности чисел. Синтаксис:
```python
for переменная in range(начало, конец):
    # код, который нужно выполнить для каждого числа в последовательности
```
Пример:
```python
for i in range(1, 6):
    print(i)
```
Циклы можно использовать для решения различных задач, таких как:

* Перебор элементов в коллекции
* Повторение блока кода до тех пор, пока не будет достигнута какая-то условность
* Создание последовательности чисел

В общем, циклы позволяют программе повторять выполнение блока кода определенное количество раз или до тех пор, пока не будет достигнута какая-то условность.
