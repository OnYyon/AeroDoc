# Установка зависимостей Backend

## Проблема: Ошибка компиляции при установке

Если при установке зависимостей возникает ошибка с Rust/Cargo, используйте один из способов ниже.

## Способ 1: Простая установка (рекомендуется)

Запустите файл:
```
install_simple.bat
```

Этот скрипт установит все зависимости без компиляции, используя предкомпилированные пакеты.

## Способ 2: Ручная установка

Откройте терминал в директории `backend` и выполните:

```bash
# Активируйте виртуальное окружение (если используете)
venv\Scripts\activate

# Обновите pip
python -m pip install --upgrade pip setuptools wheel

# Установите зависимости по одной
pip install fastapi
pip install uvicorn[standard]
pip install sqlalchemy
pip install pydantic
pip install pydantic-settings
pip install python-multipart
pip install httpx
pip install python-jose[cryptography]
pip install bcrypt
```

## Способ 3: Установка без cryptography (альтернатива)

Если проблемы с cryptography продолжаются, можно использовать упрощенную версию без шифрования:

```bash
pip install fastapi uvicorn sqlalchemy pydantic pydantic-settings python-multipart httpx python-jose bcrypt
```

Но тогда нужно будет изменить код для работы без cryptography в JWT.

## Проверка установки

После установки проверьте:

```bash
python -c "import fastapi; print('FastAPI установлен')"
python -c "import sqlalchemy; print('SQLAlchemy установлен')"
python -c "import bcrypt; print('bcrypt установлен')"
```

Если все команды выполняются без ошибок, зависимости установлены правильно.

## Если ничего не помогает

1. Убедитесь, что используете Python 3.8 или выше:
   ```bash
   python --version
   ```

2. Попробуйте установить зависимости в чистом окружении:
   ```bash
   python -m venv venv_new
   venv_new\Scripts\activate
   pip install --upgrade pip
   pip install fastapi uvicorn sqlalchemy pydantic pydantic-settings python-multipart httpx python-jose cryptography bcrypt
   ```

3. Используйте conda вместо pip (если установлен):
   ```bash
   conda install -c conda-forge fastapi uvicorn sqlalchemy pydantic python-multipart httpx
   pip install python-jose cryptography bcrypt
   ```
